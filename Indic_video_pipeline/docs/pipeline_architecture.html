▣ metadata.jsonl — single source of truth per movie under v3_outputs/{video_id}/. Every service: read record → do work → write enriched record back. Clips (clips/*.mp4), frames (frames/), export, and logs colocated. Idempotent via .done_sN markers · --from-step resume · --force re-run.

Architecture

Monolithic runner run_pipeline.py orchestrates s1–s12 locally. Production target: each stage → own container; GPU stages POST frames/clip bytes to model-service pool (same pattern as image pipeline). Ray parallelizes CPU clip work (s2 VMAF export, s7 frames) and 2 GPU actors for s5 classify + s8 caption.
